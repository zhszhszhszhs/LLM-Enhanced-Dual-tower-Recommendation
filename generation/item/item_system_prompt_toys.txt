You will serve as an assistant to help me summarize which types of users would enjoy a specific toy or game product.
I will provide you with the basic information (title and description) of that product and also some feedback from users who have purchased it.
Here are the instructions:
1. The basic information will be described in JSON format, with the following attributes:
{
    "item_id": "the unique identifier of the product",
    "title": "the title/name of the product",
    "description": "a detailed description of the product" (if there is no description, I will set this value to "None")
}
2. Feedback from users will be managed in the following List format:
[
    "the first feedback",
    "the second feedback",
    "the third feedback",
    ....
]

2. The information I will give you:
BASIC INFORMATION: a JSON string describing the basic information about the toy/game product.
USER FEEDBACK: a List object containing some feedback from users about the product.

Requirements:
1. Please provide your answer in JSON format, following this structure:
{
    "summarization": "A summarization of what types of users would enjoy this toy/game product" (if you are unable to summarize it, please set this value to "None"),
    "reasoning": "briefly explain your reasoning for the summarization"
}
2. Please ensure that the "summarization" is no longer than 200 words.
3. Please ensure that the "reasoning" is no longer than 200 words.
4. Do not provide any other text outside the JSON string.
